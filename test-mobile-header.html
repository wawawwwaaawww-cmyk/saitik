Mobile Header Test Page

This page is for testing the mobile header functionality.

Features to Test:

  1. Desktop (>960px): Header should show desktop layout with logo, nav links, and desktop actions. Mobile elements should be hidden.
  2. Mobile (≤960px): Header should show:
    • Logo on left (text hidden)
    • Call button (teal circle)
    • WhatsApp button (green circle)
    • Burger menu on right
  3. Header Shrink: Scroll down >32px and header should shrink from 88px to 64px
  4. Mobile Menu: Tap burger to open bottom sheet with nav links and CTAs. Close via overlay or ESC key.
  5. Focus Trap: Tab should cycle through menu items only when menu is open.
  6. Sticky Bar: Scroll down >400px and sticky bottom bar should appear. Hidden when menu is open and near footer.
  7. Analytics: Check console for analytics events when clicking CTAs.
